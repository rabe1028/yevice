# Security Policy

## Supported versions

Only the latest released version on `main` is supported with security updates
at this pre-1.0 stage.

## Reporting a vulnerability

If you discover a security vulnerability in `yevice`, please **do not** open a
public GitHub issue. Instead, use GitHub's
[private vulnerability reporting](https://github.com/rabe1028/yevice/security/advisories/new)
feature to report it privately.

We will acknowledge receipt within 7 days and aim to provide a fix or
mitigation within 30 days, depending on severity.
